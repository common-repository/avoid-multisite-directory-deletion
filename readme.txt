== Avoid Multisite Directory Deletion plugin will allow you to avoid directory deletion when you delete a site in wordpress multisite


== Installation ==

1. Install the plugin by accessing the Plugins section in your WordPress administrator (located in the left sidebar menu). You can also directly download it here and then upload the plugin folder to your /wp-content/plugins/ directory.2. Activate the plugin and after activation now when you delete any subsite from your multisites it will not delete the directory folder in \wp-content\uploads\sites and also all the content will be saved from deletion
